# Clipboard Image Gallery

A modern web application that allows you to easily paste and manage images from your clipboard. Built with HTML, CSS, and JavaScript.

## Features

- Paste images directly from clipboard
- Drag and drop image support
- Image gallery with preview
- Local storage persistence
- Image details view
- Search functionality
- Responsive design

## Usage

1. Copy an image to your clipboard
2. Paste it into the designated area (Ctrl+V / Cmd+V)
3. View and manage your images in the gallery
4. Search through your images using the search bar

## Development

To run this project locally:

1. Clone the repository
2. Open `index.html` in your browser
3. Start pasting images!
